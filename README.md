# CryMine Dashboard

Dashboard to monitor a variety of low-end crypto mining devices, commonly known as "lotto" miners.

## Device Support

NMMiner ESP32-based devices
NMMiner Axe & Axe Gamma devices
Lucky Miner LV-series devices
Bitaxe Gamma devices
